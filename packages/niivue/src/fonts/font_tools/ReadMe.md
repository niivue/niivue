# Usage

1. run extract_chars.bat to extract characters from your own .ttf file to a .txt file, like all_characters.txt.
2. run convertX.bat to generate .png and .json files for your .ttf font file.
3. copy the .png and .json files to the fonts folder in your projects and update your own manifest.json file.
