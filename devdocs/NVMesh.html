Constructor

new NVMesh(pts, tris, nameopt, opacityopt, visibleopt, gl, connectome)

Description:
  • a NVImage encapsulates some images data and provides methods to query and operate on images
Source:
  • nvmesh.js, line 97
Properties:
Name Type Description
rgba255 array the base color of the mesh. RGBA values from 0 to 255. Default is white
dpg array Data per group for tractography, see TRK format. Default is null (not tractograpgy)
dps array Data per streamline for tractography, see TRK format. Default is null (not tractograpgy)
dpv array Data per vertex for tractography, see TRK format. Default is null (not tractograpgy)
Parameters:
Name Type Attributes Default Description
pts array a 3xN array of vertex positions (X,Y,Z coordinates).
tris array a 3xN array of triangle indices (I,J,K; indexed from zero). Each triangle generated from three vertices.
name string <optional>
 '' a name for this image. Default is an empty string
opacity number <optional>
 1.0 the opacity for this image. default is 1
visible boolean <optional>
 true whether or not this image is to be visible
gl WebGLRenderingContext WebGL rendering context
connectome object specify connectome edges and nodes. Default is null (not a connectome).
